Welcome to CalendarGO!

Your app, called "CalendarGO," is designed for students and teachers. It provides various features for both groups. Students can sign up for classes, request additional classes, chat with teachers, pick dates, cancel classes, download an ICS file, and check tasks and upload solutions. On the other hand, teachers can manage their student groups, accept or reject classes, and check students' answers.

The technologies used in the app include Go for the backend, PocketBase for the database, and Echo as the web framework. For the frontend, React Next.js and Tailwind CSS are used. Continuous integration is handled by GitHub Actions, and the development environment is set up using Docker.

To start the app, you need to run the setupsdk.sh script, initialize and switch to the Calendar workspace, and install the required dependencies. Unit tests can be executed using the go test command, and the app can be built and served using go run Main.go serve.

Overall, CalendarGO provides a platform for students and teachers to manage classes, communicate, and track tasks and solutions. It is a great tool for both groups to stay organized and up to date with their classes and tasks.
